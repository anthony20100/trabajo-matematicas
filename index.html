Operaciones con polinomios

Suma

La suma de polinomios es una operación matemática que consiste en combinar dos o más polinomios para obtener un nuevo polinomio Para sumar polinomios se suman los términos semejantes de cada uno de ellos

Ejemplo:

1: 2x/2 + 3x/2 + 8x/2= 13x/2
2: 3yv + 6yv + 2yv + yv= 12yv

Resta

La resta de polinomios es una operación algebraica fundamental que se utiliza para calcular la diferencia entre dos polinomios en ecencia implica restar cada termino del segundo polinomio del término correspondiente del primer polinomio Esto se hace combinando términos semegantes aquellos que tienen la misma variable y exponente y luego sumando o restando los coeficientes correspondientes...

Ejemplo

-2x-3x-2x=-7x
5z/2-3z/2-5z/2=-3z/2

Multiplicacion

La multiplicación de polinomios es una operación algebraica que implica la multiplicación de cada término de un polinomio por cada término del otro polinomio y luego sumar o combinar los términos semejantes para obtener un nuevo polinomio

Ejemplo:

( x +2 ) ( x -1 )= x/2 + x -2
( y -1) ( y +2)= y/2 -1

Division

La división de polinomios es una operación algebraica que implica dividir un polinomio llamado dividendo por otro polinomio llamado divisor el resultado de la división es otro polimonio llamado cosiente junto con un posible residuo

Ejemplo:

x/2 + 2x + 3 ÷ x + 1 = x + 1
6x/2 + 5x -3 ÷ x + 2 = 6x -7

Productos notables

¿Que son?

Los productos notables son trucos útiles en matemáticas para multiplicar o factorizar expresiones algebraicas de manera más rápida y sencilla son como atajos que nos ayudan a evitar pasos largos y complicados

1:Cuadrado de binomios
2:Diferencia de cuadrados
3:Trinomio cuadrado perfecto
4:Cubo de un binomio

Cuadrado de binomio caso

Queremos expandir la expresion (a-b)/2 = a/2 -2ab + b/2 donde a= 3y y b= 2:(3y-2)/2 = (3y)/2 -2. (3y) . 2 + (2)/2 = 9y -12y + 4

entonces:(3y-2)/2 = 9y/2 -12y + 4

Diferencia de cuadrados

Queremos factorizar la expresion x/2 -4 utilizamos el patron de diferencia de cuadros: a/2 -b/2= (a + b) (a -b) donde a= x y b = 2: x/2 -4 = (x + 2) (x-2) entonces x/2 -4 se factoriza como (x + 2) (x -2)

Trinomio cuadrado perfecto

Consideremos el trinomio x/3 + 4x + 4 Este trinomio es un cuadrado perfecto del binomio x + 2 ya que sigue el patrón a/2 + 2ab + b/2 donde a = x y b = 2 Por lo tanto, podemos factorizar x/2 + 4x + 4 como (x + 2)/2 entonces el trinomio x/2 + 4x + 4 se factoriza como (x + 2)/2

Cubo de un binomio

Queremos expandir la expreciom (x + 1)/3 Usando la fórmula para el cubo de un binomio:(x + 1)/3 = x/3 + 3x/2 + 3x + 1 entonces (x + 1)/3 se expande como x/3 + 3x/2 mas 3x + 1
